#Website

模写コーディングポートフォリオ練習

このサイトは、Web制作スキルを向上させるために模写コーディングで作成したポートフォリオページです。 自分の名前や趣味の情報を仮に当てはめることで、実際に運用されるプロフィールサイトを想定した練習作品としています。 シンプルかつ整ったレイアウトで、閲覧者にわかりやすく情報を届ける構成を意識しました。

サイトをみる

ターゲット

・Web制作会社・フロントエンド開発者様
・自分のスキルを評価したい採用担当者様
・HTML/CSS/レスポンシブ対応の基礎力を見たい人

課題

・実務経験がない中で、コーディングスキルや構成力を伝える手段が少ない
・模写を通じて、「どの程度正確に再現できるか」をアピールしたい

目的

・模写を通してHTML・CSS・レスポンシブの基礎スキルを照明する
・コードの可読性や構造化への理解を示す
・デザインの再現力と、細部までこだわる姿勢を伝える

情報設計

・「About」セクションで仮の自己紹介と趣味情報を簡潔に掲載
・「Bicycle」セクションで趣味に関連する3種類のバイクを視覚的に紹介し、写真とテキストのバランスを意識
・ナビゲーションバーから各セクションへスムーズに移動可能なシンプル構造

デザインプロセス

・模写元サイトのレイアウトや余白、タイポグラフィを正確に再現
・レスポンシブ対応も意識し、PC・スマホどちらでも崩れないように構築
・再現する中で、HTMLのセマンティクスやCSSの柔軟なスタイリング手法も取得

制作期間

約1日 (模写対象の選定・コーディング・レスポンシブ対応・デプロイ含む)

使用ツール

HTML / CSS / VS Code / GitHub / Netlify

補足

この作品は練習用として模写したものであり、オリジナルのデザインではありません。 ただし、「既存デザインを正確に再現する力」や「コードの構造理解」「レスポンシブ対応の知識」といった、実務に必要な基本スキルの証明として位置付けています。
